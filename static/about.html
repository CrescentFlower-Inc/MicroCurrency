About

What exactly is MicroCurrency?

Microcurrency is a bot specifically made for micronations on discord or other mock governments who want to have a reliable economical system.
Microcurrency offers features that other often used applications do not have.

What does it have to offer over something like UnbelievaBoat?

UnbelievaBoat is a popular pick for micronations that want to have a digital economy.
However, MicroCurrency offers features that fits the requirements of a digital economy better.
For example, MicroCurrency has support for decimal values while UnbelievaBoat does not.
MicroCurrency also offers an API for developers to use when automating transactions.

Exchange

Probably the biggest benefit of MicroCurrency is that it has exchange functionality.
With the exchange features, you are able to easily exchange between different currency.
The exchange rates are also dynamic, meaning that whenever somebody exchanges one currency for another, the exchange rates change.
This opens up the possibility for trading currencies.

Open-source

MicroCurrency is completely opensourced under the GPL-2 License.
This means that if you have concerns about the global micronation economy becoming too centralized around Microcurrency, you can just run your own instance on your own terms.
Another benefit of open-source is that the community can make contributions and that people can modify their own instance of microcurrency.
Unfortunately, if MicroCurrency becomes too decentralized, exchanging between currencies becomes harder and harder.
This is why we are considering making MicroCurrency into a federative application, meaning the application is split among several bots run by other people but still connected
